# 📊 Sales Analytics Dashboard – Apache Superset

## 📌 Project Overview
This project presents an interactive Sales Analytics Dashboard built using Apache Superset.  
The dashboard provides insights into revenue performance, customer distribution, product contribution, and regional sales trends.

It is designed to help business stakeholders monitor KPIs and make data-driven decisions.

---

## 🎯 Key Performance Indicators (KPIs)

- 💰 Total Revenue: 21.8K
- 📦 Total Quantity Sold: 312
- 🛒 Average Order Value: 2.72K

---

## 📈 Dashboard Components

### 1️⃣ Daily Sales Trend
- Line chart showing sales quantity trend over time
- Helps identify sales peaks and drops

### 2️⃣ Revenue by Province
- Bar chart comparing revenue across provinces
- Bagmati shows highest revenue contribution

### 3️⃣ Customer Type by Province
- Comparison between Retailer, Wholesaler, and Distributor
- Helps understand regional customer segmentation

### 4️⃣ Product Share
- Pie chart showing product contribution:
  - Rice (Highest share)
  - Oil
  - Sugar
  - Flour

### 5️⃣ Top Customers by Revenue
- Retailer and Wholesaler generate highest revenue
- Distributor contributes smaller share

---

## 🛠 Tools & Technologies

- Apache Superset
- SQL for data querying
- Data aggregation & KPI calculation
- Interactive data visualization
